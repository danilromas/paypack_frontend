PayPack Marketplace — Chrome extension (developer mode)

1) Download paypack-marketplace-extension.zip from your PayPack site:
   /api/extension/download/

2) Unzip to a folder (e.g. paypack-marketplace).

3) Chrome → chrome://extensions → enable "Developer mode".

4) "Load unpacked" → select that folder.

5) Click the extension icon → your PayPack site URL defaults to https://paypack.uno (change it there if you use another deployment).

6) Open a Facebook Marketplace listing — use "Buy in PayPack" on the listing page (or copy import link from the extension popup).

For updates: remove the old extension, unzip the new zip, Load unpacked again.
